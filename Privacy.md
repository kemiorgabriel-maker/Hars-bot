# Política de Privacidade — Hars Bot

A privacidade dos usuários é uma prioridade para o Hars. Esta política descreve quais dados coletamos e como os tratamos de forma transparente.

1. Dados que Coletamos
Para funcionar corretamente, o Hars armazena apenas as informações públicas e estritamente necessárias fornecidas pela API do Discord:
- IDs de Usuários, IDs de Servidores e IDs de Cargos (necessários para salvar configurações e preferências).
- Textos de comandos ou interações enviados diretamente ao bot.

2. O que NÃO Coletamos
O Hars NÃO lê, armazena ou monitora o histórico de conversas dos servidores, dados pessoais (como e-mails ou senhas) ou qualquer informação fora do escopo de seus comandos.

3. Compartilhamento de Dados
Nenhum dado coletado pelo Hars é vendido, alugado ou compartilhado com terceiros. As informações são utilizadas exclusivamente para a execução das funções do próprio bot dentro do Discord.

4. Exclusão de Dados
Os dados de um servidor são automaticamente elegíveis para exclusão quando o bot é removido (kick/ban) dele. Caso um usuário queira solicitar a remoção manual de qualquer dado associado ao seu ID do Discord, basta entrar em contato com o desenvolvedor responsável.
